# Practical 2 Part 1: Thinking Back and Looking Forward

## Summary

This part involves reflection on previous practical work and introduces the investigation into background radiation using a cloud chamber.

## Key Points

### Practical Science Skills

**Important aspects of practical science:**
- Planning investigations
- Making accurate observations
- Recording data systematically
- Analyzing results objectively
- Interpreting findings
- Drawing conclusions from evidence

**Skills developed:**
- Using scientific instruments
- Recording observations accurately
- Creating graphs and tables
- Writing scientific reports
- Critical evaluation of data

### Investigation Introduction

The main investigation focuses on:
- Physics of natural radioactivity
- Using a cloud chamber
- Measuring background radiation levels
- Determining radiation types from unknown samples

### Scientific Communication

Scientists must communicate effectively through:
- Oral presentations
- Written reports
- Poster summaries
- Explaining to journalists/public

### Peer Review

Essential part of scientific enterprise:
- Providing constructive feedback
- Receiving feedback to improve work
- Supporting others' development
